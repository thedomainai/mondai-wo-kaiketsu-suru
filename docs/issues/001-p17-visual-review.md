# P17 Visual Review

## 問い

P17「今日のテーマ」は、問題解決とロジカルシンキングを接続する橋渡しのスライドとして、何を改善すべきか。

## 論点

### 1. Layout

- 上段と下段でカラム設計が揃っていない。
- 左右のボックスが独立して見え、1つのストーリーとして流れていない。
- 中央の「その手法の1つが」が、上下をつなぐ補助記号としては弱い。

### 2. Typography

- 主メッセージの存在感が強すぎて、他の要素との階層差が開きすぎている。
- 右上の `PROBLEM SOLVING` カードと右下の結論カードの文字サイズ差・重み差が中途半端で、役割の違いが曖昧。
- 改行位置が「意味のまとまり」より「箱の都合」に寄っている。

### 3. Graphics

- 左上の主メッセージ枠、右上カード、中央ピル、左下黒カード、右下説明帯でオブジェクトの文法がバラバラ。
- 線・矢印・ピル・黒ベタの併用で、視覚言語が散っている。
- 接続を示したいのか、比較を示したいのかが混線している。

## 論点分解

### Layout の問題の正体

1. 上段は `左 688px / 右 flex`、下段は `左 520px / 右 flex`
2. 左右の基準線が上下で一致していない
3. 中央ピルが「上段のまとめ」なのか「下段への接続」なのか曖昧
4. 右上カードだけが端に寄って孤立して見える

### Typography の問題の正体

1. 主メッセージだけ太く大きく、他のテキストが追従できていない
2. `問題解決` と `論理的思考` の見出しの関係が、並列なのか上下関係なのか見えにくい
3. 本文コピーが少し説明的で、密度の割に推進力が弱い

### Graphics の問題の正体

1. 枠線カード
2. 灰色カード
3. 黒ベタカード
4. ピル
5. 線と矢印

これらが1枚の中に並び、造形ルールが散っている。

## 仮説構築

### 案A

上下2段は維持しつつ、同じ2カラムグリッドに揃える。

- メリット: 現行構成を活かしやすい
- デメリット: 中央接続の曖昧さが少し残る

### 案B

左から右へ1方向に流れる1本のストーリーに組み替える。

- `今日のテーマ`
- `問題解決`
- `論理的思考`
- `この研修で扱う意味`

- メリット: 橋渡しスライドとして最も自然
- デメリット: 現在の上下構成を作り直す必要がある

### 案C

左に主メッセージ、右に「問題解決 / 論理的思考 / 接続理由」を縦積みで整理する。

- メリット: 情報階層が明確
- デメリット: 少し説明スライド寄りになる

## 仮説の評価

| 案 | 橋渡しの明確さ | 視線誘導 | 実装負荷 | 評価 |
| --- | --- | --- | --- | --- |
| A | 中 | 中 | 低 | 保守的 |
| B | 高 | 高 | 中 | 最有力 |
| C | 高 | 中 | 低 | 安定 |

## 結論

P17で最も改善すべきは、**上下で揃っていない構造を、1本の流れとして再設計すること**。

優先順位は以下。

1. 上下で異なるカラム設計をやめる
2. 中央ピルと線の役割を整理する
3. オブジェクト文法を減らす
4. 主メッセージ以外の階層を再設計する

## ネクストアクション

1. レイアウト方針を `案B` か `案C` で決める
2. 接続オブジェクトを1種類に絞る
3. `問題解決` と `論理的思考` の関係を一目で読める配置にする
4. コピーの改行位置を意味単位で調整する
