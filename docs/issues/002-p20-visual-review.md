# P20 Visual Review

## 問い

P20「再現性とは『正しく間違えること』」は、ロジックツリーの構造と探索の読み筋を、どこまで迷わず読める状態にできているか。

## 論点

### 1. Layout

- ツリー全体が少し上寄りで、パネル下側の空白が勝っていた。
- 終点ノード付近の距離感が窮屈で、最後の読み筋が詰まって見えていた。
- 比較対象の枝と、次に進む枝の関係が一目で拾いにくかった。

### 2. Typography

- 終点ノードの文字が箱に対してやや窮屈だった。
- ゴースト側のラベルが薄すぎて、比較対象としての役割が弱かった。

### 3. Graphics

- 折れ線化はできていたが、線の濃淡差が強すぎて一部が消えかけに見えた。
- 失敗した枝と進む枝の違いが、線と箱の両方で十分に補強されていなかった。

## 結論

優先すべき改善は以下。

1. ツリー全体を少し下げて、パネル内の重心を中央に戻す
2. ゴースト枝を「消す」のではなく「比較対象として残す」濃度に調整する
3. 終点ノード周辺の余白を増やし、最後の読み筋を詰まらせない
4. 失敗枝と進行枝を、線色と面色の両方で読み分けやすくする

## 実施

1. ノード群と接続線を下方へ再配置
2. 折れ線のストローク濃度を再調整
3. ゴーストノードの不透明度を引き上げ
4. `a-1-1` と `a-1-2` に淡い面色を追加
5. 終点ノードの幅と接続余白を調整
