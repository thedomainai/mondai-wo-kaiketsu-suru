# スライド構成変更計画

## 新しいスライド構成のルール

```
## header
- 章タイトル        → data-header-chapter（既存）
- メインメッセージ  → data-header-subtitle（既存）を昇格
- スライドタイトル  → data-header-title（既存）

## body
- ディスクリプション: メインメッセージの補足
- メインコンテンツ: 図解など

※ 最下部の締め文・格言・結語テキストは削除
```
