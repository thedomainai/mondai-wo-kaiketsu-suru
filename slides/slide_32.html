01

売上を伸ばしたい

問いは「売上をどう伸ばすべきか」になる。分解は客数 x 客単価、さらに新規 / 既存、単価 / 回数など、動かす変数へ下りていく。

分解例

客数 / 客単価 / 流入別 / 既存比率

02

状況を見やすくしたい

問いは「どこを見える化すべきか」になる。分解は商品別、顧客別、地域別、チャネル別など、切り口をそろえて状況把握しやすくする方向へ向かう。

分解例

商品 / 顧客 / 地域 / チャネル

03

利益構造を管理したい

問いは「どの変数を管理すべきか」になる。分解は売上 - 原価 - 固定費、さらに粗利率や回収効率など、利益を左右する変数に降りる。

分解例

売上 / 原価 / 固定費 / 粗利率

同じテーマでも、目的が違えば問うべきことも分け方も変わる。
