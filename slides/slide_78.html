第10章 まとめ

まとめ

前提から仮説構築までの要点

CORE MESSAGE

正しい問いを立て、正しく分解し、仮説を置いて検証し、受け手に伝える。
この一連の精度が、行動の期待値を押し上げる。

PREMISE

行動と結果は切り離す

再現性とは、正しく間違えながら成功確率を高め続けられる状態である。

OVERVIEW

問題解決とは行動を決めること

論点に解を出し、次の一手を定めることが問題解決の本質である。

ISSUE

問いの質が勝敗を決める

問いを誤れば、その後の分解も仮説も検証もすべてずれる。

DECOMPOSITION

1階層ずつ、MECEに分解する

上下の階層を行き来しながら、構造の漏れと重なりを消していく。

HYPOTHESIS

仮説は順序と再現性を生む

正解を当てるためではなく、何を検証しどう修正するかを明確にするために置く。

FLOW

論点定義 → 分解 → 仮説 → 検証 → 伝達

この流れを雑にしないことが、仕事の期待値を上げる最短ルートである。
